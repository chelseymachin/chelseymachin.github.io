NextJS Pokedex Front-End

HTML CSS JavaScript NextJS
Github Repository

Summary: This fun little project is just a simple and polite front-end representation of the popular Pok√©API (RESTful API for classic pokemon data). In a simple overview, it fetches data from the API and pushes and presents it in a pretty way in the program. There's some light effects added, but really this project is just to show off that I can consume RESTful APIs and present data!

Tech Specs: I used NextJS as a framework to build out a single page, static web application. I utilized hooks from the SWR library to fetch the data from the API. There's a component that receives the name of the pokemon, which is then used to fetch the correct Pokemon's data. This component then displays the Pokemon information on page load.
