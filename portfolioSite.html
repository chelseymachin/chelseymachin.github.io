Portfolio Site

HTML CSS JavaScript SASS Bootstrap
Github Repository

Summary: No video for this one because we're currently exploring it! This site is a platform to display my projects, professional history, and also give potential employers a chance to get to know me and download my resume. There's links to my different social media platforms (LinkedIn and Github), and a contact link to send an email directly to my designated email address. It's fully responsive, and developed mobile-first. There are .gifs representing brief use of the projects on the portfolio, and separate landing pages for each of them.

Tech Specs: This site was inspired by some HTML templates I saw online and liked the look of. It's written in your basic HTML/CSS/JavaScript, with some added SASS scripting. The HTML is semantic and easy to read, and the CSS is designed to be similarly structured. While my forte isn't web design, I'm pretty proud of the aesthetic I was able to achieve on this site. It's hosted through the Github page publication feature (because it's free!).
